How to run the Game
-------------------
Just run:
    game.exe

System Requirements
-------------------
 * 8086 / 8088 CPU
 ** 80286 10MHz recommended
 * 640kb of RAM
 * VGA/MCGA graphics card, or compatible
 * Adlib soundcard, or compatible (OPTIONAL)
 * Soundblaster soundcard, or compatible (OPTIONAL)
